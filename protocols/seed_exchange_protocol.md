# Seed Exchange Protocol

## Purpose

To provide a simple, traceable, and stewardship-centered process for seed exchange across local and regional nodes.

## Principles

- share with care
- preserve provenance
- note adaptation context
- communicate uncertainty honestly
- protect local ecological knowledge
- avoid unnecessary barriers to exchange

## Minimum exchange record

Each exchange should ideally include:
- accession identifier
- common and scientific name
- stewardship status
- year of seed saving
- pollination notes
- adaptation notes
- sender
- receiver
- exchange date

## Human norms

Seed exchanges should favor:
- reciprocity
- documentation
- local adaptation awareness
- respect for steward effort
- generosity without loss of traceability

## Machine-readable compatibility

Where possible, exchanges should emit:
- a stewardship event
- accession linkages
- provenance metadata
