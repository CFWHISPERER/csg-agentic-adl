# Seed Infrastructure Protocol Roadmap

Seed Infrastructure Protocol (SIP)  
Version: 0.1-draft  
Status: Evolving Specification

---

## Overview

This roadmap outlines the planned structure of the Seed Infrastructure Protocol
specification.

The protocol is being developed incrementally. Some documents are already
implemented, while others represent future components of the architecture.

The goal is to define an open framework for coordinating distributed seed
stewardship networks across bioregions.
