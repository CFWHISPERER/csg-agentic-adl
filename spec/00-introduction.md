# Seed Infrastructure Protocol (SIP)

The Seed Infrastructure Protocol defines a framework
for coordinating distributed seed stewardship networks.

Its goal is to support ecological resilience through
decentralized stewardship and open knowledge exchange.

## Core Objectives

• preserve agricultural biodiversity
• support local adaptation
• enable seed sharing networks
• coordinate stewardship across bioregions

The protocol defines standards for:

• seed data schemas
• stewardship events
• governance decisions
• agent coordination
• network federation
